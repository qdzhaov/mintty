Running mintty

C

运行mintty

E First off, instead of installing mintty manually, consider installing it through Cygwin's setup.exe, where it can be found in the Shells category. This will place a shortcut to mintty in the Cygwin folder of the start menu. C 首先，与其手动安装 mintty，不如考虑通过 Cygwin 的 setup.exe 安装它，在那里可以在 Shells 类别中找到它。这将在开始菜单的 Cygwin 文件夹中放置 mintty 的快捷方式。

E Otherwise, you need to ensure that Windows is able to find the Cygwin DLL (cygwin1.dll) when starting mintty.exe. There are several ways to do this: C 否则，您需要确保 Windows 在启动 mintty.exe 时能够找到 Cygwin DLL (cygwin1.DLL)。有几种方法可以做到:

  • E Start it from within an existing Cygwin session. C 从现有的 Cygwin 会话启动它。
  • E Place mintty.exe alongside cygwin1.dll in the Cygwin /bin directory and start it from there. C 将 mintty.exe 与 cygwin1.dll 放在 Cygwin /bin 目录中，然后从那里启动它。
  • E Add the Cygwin bin directory (normally C:\cygwin\bin) to the Windows PATH variable, which can be found under 'Environment Variables' on the 'Advanced' tab of the System Properties control panel. C 将 Cygwin bin 目录(通常为 c: Cygwin bin)添加到 Windows PATH 变量中，可以在 System Properties 控制面板的“ Advanced”选项卡中的“ Environment Variables”下找到该变量。
  • E Create a shortcut to mintty.exe with the working directory set to the Cygwin bin directory. This can be done with the create_shortcut.js script. C 创建一个快捷方式到 mintty.exe，并将工作目录目录设置为 Cygwin bin 目录。这可以通过 create _ shortcut 完成。Js 脚本。

E Invoke mintty with a dash as its only argument to obtain a login shell that sources all the usual startup scripts: mintty - C 以一个破折号作为它的唯一参数来调用 mintty，以获得一个登录 shell，它为所有常见的启动脚本提供源代码: mintty-

E Invoking mintty with --version or --help will print version or usage information. The mintty manual can be found on the home page at http://mintty.github.io/. C 调用 mintty with -- version 或 -- help 将打印版本或使用信息。明尼苏达州的手册可以在 http://mintty.github.io/的主页上找到。
